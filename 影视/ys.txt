APP功能与操作说明

# 通用(特殊)功能
* 版本号 点按正式版更新 长按内测版更新
* 單條點播站源或直播列表配置可指定播放器
   (添加"playerType": 0,1,2   0:系统播放器 1: IJK   2: EXO 不寫則默認跟隨接口設定)
* 本地接口兼容TVB接口协议 自动转 file:// 协议
   (clan://localhost/  和   clan://)
* 本地接口配置可浏览模式直接选中接口文件
* 支持多线路接口识别
   写法示例：
{"urls":[
{"url":"http://饭太硬.top/tv","name":"饭总"},
{"url":"http://xhdwc.tk/0","name":"骚零"}
]}
* EXO 支持帶驗證的地址
* 支持油管直播点播解析
   (網址要包含www.youtube.com才會解
    大陸地區需要翻墻)
* 支持缓存清理
* 支持DOH
  (即dns服务 可接口写入)
* EXO: 隧道模式 连线方式 ua模式
   ( APP设置 播放器切换到EXO 
     然后长按即可弹窗
     隧道模式即原音原画直通輸出模式 
     必須使用 SurfaceView 需要高级设备支持 
    一般不需打开此功能)
* IJK:ua模式
   (类同EXO操作)
* 自动切换播放器
* 支持荐片协议地址(jianpian://)
* 支持超时时间设定
  (添加参数"timeout": 30 单位：秒)
* 支持字幕大小设定*ssa格式暂不支援
  (设置长按播放器全局设定
   播放頁设定:
   電視版菜單移動至字幕，上下按鍵調整
   手機版長按字幕，彈出菜單調整 )
* 支持本地文件代理地址写入配置
  (比如: http://127.0.0.1:9978/file/tvbox/token.txt
   文件路径即是：主目录/tvbox/token.txt )

# 点播功能
* 支持投屏
   (手机端投＞TV端&接口一同投送和支持DLNA接收的设备)
* 內置聚合解析
* 支持語音搜索
* 支持 HTTP/3&smb 協議
* 支持略過廣告切片/支持rules規則
   (20230518內測版起殼不再自動略過廣告 
     需寫切片廣告規則到接口配置食用
     注意:此規則配置暫不能兼容TVB等APP)
* 支持鄰近裝置互動
* 軌道切換自動記憶
* 片頭片尾快捷標記與微調(手機端無微調)
   (TV端方向上键+OK.以影片时间一半前后
    為標準设定片头/片尾 對焦後上下鍵微調)
* 刷新、重播一鍵切換
   (TV端独有 长按切换 点按实现当前功能 
    手机端详情页下拉刷新)
* TV端支持分類刷新
   (焦點分類上鍵刷新 老版本 長按刷新)
* 自動換線路，換站源
   (可配置設定關閉換源 添加"switchable": 0 
    配置載入過久可返回也可APP設定關閉)
* 收藏的影片可自動切換至該接口
* 採用最新MD3和影魔IJK播放器支持軟硬解
* 支援調用idm+下載影片
   (手機版獨有功能 詳細頁播放器窗口分享按鈕 >選擇idm+即可下載)
* 全屏狀態下支持選集播放
   (手機版詳情頁選集後面的"更多"按鈕也可選擇更多的片集)
* 支持單集播放循環
   (需播放器菜單欄開啟)
* 支持列表模式顯示
   (alist one等網盤和本地視頻體驗最佳)
* 配置載入過久可返回
* 搜索支持文件夾格式
* 支持同步收藏和觀看紀錄
  (20230606版起覆蓋同步 請無視下面規則)
  (需手機和電視更新到最新版 )
  同步規則(接口配置一同同步):
   1. 雙向同步
       (手機 <—> 電視
        手機 <—> 手機
        電視 <XX>電視)
   2. 同一部片新的蓋掉舊的
       (比如手机昨天放第二集 
         今天电视播放第一集 
         那么同步同为第一集)
   3. 不存在則新增紀錄同步規則
* 支持手动换源
   (手机版详情页长按片名复现)
* 支持假熄屏
   (手机版独有:画中画模式可熄屏播放)
* 支持js，py本地代理
* js 支持外掛 jar
* 支持【貓影視】js 格式
* TV端支持站點觀看記錄無痕設置
  (设置 >小房子)
* TV端站点支持多列显排设置
  (同上操作)
* 搜索支持翻页
   (选择支持多页搜索的站源一目了然)
* 支持站点无痕播放
  (接口站点配置可添加"recordable":0参数复现 也可APP设定)
* 无播放记录时支持(线路)手动播放
  (设置栏播放器长按设置)
* 播放音频支持显示广播封面
* 支持多視圖模式
   (站點添加"viewType": 0-5)
     0:默認(豎圖)，1:橫圖，2:圓圖，
     3:海報，4:橫向單列中圖帶片名，
     5:橫向單列小圖帶片名
       (當圖片無法加載時其他模式也顯示該模式)
   从20230807#1内测版起 食用下面参数 
   直版 "style":{ "type":"rect" }
   橫版 "style":{ "type":"rect", "ratio":1.33 }
   正方 "style":{ "type":"rect", "ratio":1 }
   正圓 "style":{ "type":"oval" }
   橢圓 "style":{ "type":"oval", "ratio":1.1 }
* 电视端支持清除所有历史歷史
   (紀錄刪除狀態下 再次長按)
* 0827起 支持貓的預設篩選及資料夾

# 直播功能
* 带密码分组隐藏
   (狂点收藏5次弹框输入密码后才能显示分组
    手機版20230816#2起支持指紋解鎖顯示分組
    1.7.8版起 配置添加"pass" : true
    可以直接显示密码分类
    已收藏的密码分类的频道也可显示食用)
* 支持多条直播源列表(俗称直播接口)
* 配置支持常规txt m3u格式和影视独有的jso途同归n格式直播列表
   (即配置"type": 0和1的区别)
* 配置可設定 EPG 地址
   (节目信息和logo)
* 配置可設定啟動 App 進入直播
   (添加 "boot": true)
* 支持 p2p~p9p、mitv、zlive、tvbus 等特殊協議
   (64位APP不支持)
* 配置支持header/ua/referer
   (个别源需要添加此项一種或多鐘方可正常播放)

# 推送功能
* 内置直链推送服务
  (无接口配置直链也可推送播放)
* 手机版支持粘贴板识别
  (也可自行选择粘贴板)
* 手机版支持本地视频播放
* 支持推送 迅雷全家桶链接和http種子
